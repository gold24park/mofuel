# Color Index

The following is a list of every hex code that corresponds to a custom color setting:

| [7234BF](https://www.color-hex.com/color/7234BF) | item_destroy_text | The text color of the numbers next to Essences as they appear to the left/right of the reels, denoting the number of times they must be destroyed.  Referred to as `Item Text 2` in the settings menu. |
