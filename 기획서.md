## 게임 개요

- **장르:** 로그라이크 덱 빌딩 / 다이스 포커 / 실시간 텐션
- **플랫폼:** PC / Mobile
- **핵심 컨셉:** 은행을 턴 후, 경찰의 추격을 따돌리고 베이스 캠프까지 무사히 도주하는 하이 텐션 다이스 레이싱

---

## 플레이어 캐릭터

게임 시작시 갱단을 창설하고 한 명의 캐릭터를 선택하며, 각 캐릭터는 플레이 스타일에 영향을 미치는 고유한 특성을 가지고 있다.

**예시:**

- **Driver:** 거리 환산 보너스
- **Thug:** 더블 다운 점수 배율 상승
- **Mastermind:** 리드로우 1회 추가

---

## 핵심 게임 루프

1. **은행 선택:** 목표 거리와 난이도, 특성이 다른 은행 중 하나를 고른다. 캐릭터 특성과 은행 특성마다 다르지만 기본적으로 타이머 7초가 주어지고 도주를 시작한다.

2. **주사위 드로우 (시간 정지):** 인벤토리에서 무작위로 뽑은 주사위 8개가 Hand에 들어온다. 플레이어는 주사위간의 시너지를 고려하여 Hand에서 5개의 주사위를 고른다.
   - **리드로우:** 게임당 2회 제공되며, Hand의 주사위 풀이 마음에 들지 않을때 새롭개 8개를 다시 뽑는다.

3. **굴리기 (타이머 시작):** 굴리기 버튼을 누르는 순간 타이머가 시작되고 남은 시간이 줄어든다. 주사위를 굴리면 자동으로 가장 높은 점수의 족보가 자동 채택된다. 플레이어는 이 점수로 확정 / 리롤 / 더블 다운 중 하나의 액션을 선택해야 한다.
   - **리롤:** 게임당 3회 제공. 다시 굴릴 주사위를 선택하고 주사위를 굴린다. (주사위를 고르는 동안은 시간이 정지)
   - **더블 다운:** 리롤 기회를 2회 소진하여 기본 x2배의 강력한 배율을 얻지만, 이후 수정없이 결과를 즉시 받아들어야 한다.

4. **점수 확정 (시간 정지구간):** 점수 확정시 플레이어에게 어떻게 해당 점수가 나오는지 주사위간 시너지가 일어날때, 효과가 발동될때마다 애니메이션이 재생되며 게임 쥬스를 더한다. 애니메이션이 끝나면 다시 시간이 흐르면서 플레이어가 어떻게 점수를 치환할지 선택한다. 이 버튼은 타이머와는 무관하게 3초안에 선택해야한다. 선택하지 않으면 자동으로 거리 환산으로 채택된다.
   - **거리 환산:** 점수를 소모해 베이스 캠프까지의 남은 거리를 줄인다.
   - **시간 확보:** 점수를 소모해 타이머에 시간을 추가한다.

5. **반복 및 클리어:** 2~4를 반복하여 주어진 시간 안에 목표 거리를 채우면 베이스 캠프 도착으로 클리어, 아니면 뒤쫓아오는 경찰에 체포되어 게임 오버. 클리어 후 베이스 캠프에서 돈을 정산 받고, 덱을 정비하고 트렁크에 물건을 채우면서 다음 습격을 준비한다.

## 성장 시스템: 베이스 캠프

### 상점

- 주사위 및 기어 구매
- 보유한 주사위 및 기어 판매

### 트렁크 (적재 공간)

- 4x4 사이즈(기본)의 공간에 상점에서 구매한 기어를 배치합니다.
- 각 기어는 크기와 모양이 다르며, 이를 효율적으로 배치해 공간 관리를 해야합니다.
- 각 기어는 발라트로의 조커카드처럼 고유한 패시브 효과가 있습니다.

### 로드아웃

- 트렁크에 어떤 기어를 실을지 전략적으로 선택할 수 있습니다.

### 특별 NPC

- 무작위 이벤트로 주사위 혹은 족보 업그레이드 기회 / 트렁크 확장 기회 등 제공

## 시각적 연출 및 UI

| 위치                     | 내용                                                                                          |
| ------------------------ | --------------------------------------------------------------------------------------------- |
| **화면 최상단**          | 남은 시간 타이머, 베이스 캠프까지 남은 거리를 시각화                                          |
| **화면 중앙**            | 주사위 및 족보 이펙트, 결정 버튼들                                                            |
| **화면 중앙하단 (배경)** | 도주하는 차량과 쫓아오는 경찰차의 모습 (2차원). 경찰차와의 거리는 시간이 임박할수록 좁혀진다. |
| **화면 하단**            | 주사위 핸드, 리드로우 버튼                                                                    |
| **화면 우 하단**         | 트렁크 그리드. 점수 확정시 오너먼트가 점수에 영향을 줄경우 반짝이며 게임 쥬스 제공            |

### 연출

- **긴박감 연출:** 시간이 2초 미만으로 남으면 화면이 붉게 점멸하며 사이렌 소리가 커짐.
- **결과 연출:** 성공 시 플레이어 차가 화면 밖으로 가속하며 탈출, 실패 시 반대편에서도 경찰차가와서 막아서며 붙잡힘.
